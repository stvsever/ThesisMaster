# Evaluation Directory

This directory stores pseudoprofiles, intermediate evaluation artifacts, integrated run outputs, QA tests, and research communication reports.

## Core Folders

- `00_pipeline_orchestration/`
  - End-to-end orchestration entrypoints (`run_pipeline.py`, `run_pseudodata_to_impact.py`).
- `01_pseudoprofile(s)/`
  - Free-text and pseudodata sources.
- `02_mental_health_issue_operationalization/`
  - Complaint-to-criterion mapping outputs.
- `03_construction_initial_observation_model/`
  - Initial observation model construction artifacts.
- `04_initial_observation_analysis/`
  - Legacy/manual readiness, network, and impact outputs.
- `05_integrated_pipeline_runs/`
  - New standardized run-scoped outputs generated by the integrated orchestrator.
- `06_quality_assurance/`
  - Research-grade `pytest` test framework (unit + integration + smoke markers).
- `07_research_communication/`
  - Scripts for communication-ready reporting of integrated runs.
